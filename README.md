# COMP-4400 Tutor

A RAG (Retrieval-Augmented Generation) chatbot for COMP-4400 — Principles of Programming Languages at the University of Windsor. Ask it anything from the course: lambda calculus, Scheme, Prolog, MapReduce, axiomatic semantics, code optimization, garbage collection, AOP, and more.

## Prerequisites

- Python 3.10+
- An [OpenAI API key](https://platform.openai.com/api-keys)
- Course PDFs downloaded from the course page

---

## Setup

### 1. Create and activate a virtual environment

**Mac / Linux**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows — Command Prompt**
```cmd
python -m venv venv
venv\Scripts\activate
```

**Windows — PowerShell**
```powershell
python -m venv venv
venv\Scripts\Activate.ps1
```

**Windows — Git Bash**
```bash
python -m venv venv
source venv/Scripts/activate
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Add your OpenAI API key

```bash
# Mac / Linux / Git Bash
cp .env.example .env

# Windows Command Prompt
copy .env.example .env
```

Then open `.env` and replace `sk-...` with your actual key:

```
OPENAI_API_KEY=sk-your-key-here
```

---

## Running the Chatbot

### Step 1 — Add your PDFs

Drop all COMP-4400 course PDFs into `data/pdfs/`.

### Step 2 — Build the vector store (run once)

```bash
python ingest.py
```

This parses the PDFs, splits them into chunks, generates embeddings, and saves them to `data/vectorstore/`. Only needs to be re-run if you add or change PDFs.

### Step 3 — Launch the web UI

```bash
python app.py
```

Open [http://localhost:7860](http://localhost:7860) in your browser.

## Project Structure

```
COMP4400-tutor/
├── ingest.py          # PDF → chunks → embeddings → ChromaDB
├── tutor.py           # RAG chain: retrieval + GPT-4o-mini + conversation history
├── app.py             # Gradio web UI
├── requirements.txt
├── .env               # your API key (not committed)
├── .env.example       # template
└── data/
    ├── pdfs/          # put course PDFs here
    ├── raw_text/      # extracted text (auto-generated, useful for debugging)
    └── vectorstore/   # ChromaDB store (auto-generated by ingest.py)
```

## Tech Stack

| Component     | Tool                              |
|---------------|-----------------------------------|
| PDF parsing   | PyMuPDF                           |
| Text chunking | LangChain RecursiveCharacterTextSplitter |
| Embeddings    | sentence-transformers all-MiniLM-L6-v2 (local, free) |
| Vector store  | ChromaDB                          |
| LLM           | OpenAI gpt-4o-mini                |
| UI            | Gradio                            |
